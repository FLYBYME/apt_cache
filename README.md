# Project Overview
A Node.js-based API server for managing apt cache data.

## Quick Start
1. Install dependencies via `npm install`
2. Run the server with `npm start`

## Configuration
The application configuration is defined in `app.json`, which contains environment-specific settings (e.g., database URLs, API keys).

## Dependencies
Key dependencies from `package.json` include:
- `dhcp`
- `fs-extra`
- `last-one-wins`
- `mime`
- `tftp`

## Folder Structure
- `app.json`: Contains environment-specific settings and infrastructure configuration.
- `package.json`: Defines project metadata and dependencies.
- `server.js`: The main entry point and HTTP server logic for the application.
